- Binary Converter 
A simple tool and Fast for converting text to binary code and binary code to text.

I created this project to understand binary code practically and to make it easier for myself (and anyone who needs it) to experience conversion without relying on external websites or tools with annoying ads.

- Project Description
 The project is a simple page with an input field where you can write text or binary code. With the click of a button, the text is converted to binary code, or the binary code is converted to plain text. I tried to make it simple and fast without any complications.
